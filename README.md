## 功能特性与限制说明

### 可用功能
✅ **完整微信关联功能**
- 支持所有微信账号绑定操作
- 可正常使用消息收发等核心功能

### 已知限制
❌ **内置API服务不可用**
- 未激活状态下无法使用平台提供的免费API和KEY
- 解决方案：需自行配置第三方API服务（如DeepSeek、火山引擎等）

> 提示：当系统弹出API选择窗口时，请选择其他支持的云服务厂商进行配置。
